# ExGdax

GDAX API client for Elixir.

## Installation

**NOTE: This project is currently under development.**

List the Hex package in your application dependencies.

```elixir
def deps do
  [{:ex_gdax, "~> 0.1.0"}]
end
```

Run `mix deps.get` to install.
